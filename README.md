# Diseño de dispositivos para el Internet de las Cosas

## Descripción del curso

El curso Diseño de dispositivos para el Internet de las Cosas es una introducción al diseño y análisis de dispositivos para el Internet-de-Cosas. Los estudiantes estudiarán los principios de diseño de productos electrónicos enfocados al Internet de las Cosas en cada una de las etapas asociadas, comprendiendo los desafíos fundamentales en toda la cadena de producción.
